# 💼 InternHub - Internship & Career Platform

## 📌 Project Description
InternHub is a full-stack web application that helps students explore internships, courses, and assessments in one place. It is designed to simplify career development and skill tracking.

---

## 🚀 Features
- 🔐 User Authentication (Login/Register)
- 🎯 Internship Listings
- 📚 Courses Section
- 📝 Assessments & Skill Tests
- 📄 Resume Upload
- 📊 Dashboard for users

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Django
- Database: SQLite
- Tools: VS Code, Git
